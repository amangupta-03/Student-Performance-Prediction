# Student Performance Prediction Using Machine Learning

This project predicts whether a student will pass or fail using academic and demographic data.

## Dataset
Kaggle Student Performance Dataset (student-mat.csv)

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Models Used
- Decision Tree
- Random Forest

## Result
Random Forest achieved the highest accuracy (~88%).

## Conclusion
The model successfully predicts student performance and can help identify at-risk students.
